  • Collect basic project information

  • Sign offer and send it back to us
  • Submit the sample sheet to us
  • Await sequencing analysis
  • Access results via portal

What we need from you
  • Project aim
  • Number of samples
  • Number of replicates
  • Sample type (library, DNA/RNA, blood, stool,...)
  • Technology (Illumina/Pacbio/Nanopore/10x)
  • Application (WGS, exome, metagenome,...)
  • Sequencing depth (coverage or output in Gb/sample or Mio Reads/sample)
  • Bioinformatics needed? If yes, how much support?
